## LinkedIn

We just released Appwrite MCP Server 2.0!

The Appwrite MCP server connects your Appwrite project to AI coding agents like Cursor, Claude Code, Codex, VS Code, and more through the Model Context Protocol.

In the previous version, you had to pass flags like --users or --storage to enable specific Appwrite services. The more services you enabled, the more tool definitions were loaded into the model's context, leaving less room for your actual prompts and code.

With Appwrite MCP Server 2.0, that changes. The server now supports all Appwrite services automatically. Instead of registering dozens of tools with the model, it exposes exactly two: one to search the Appwrite tool catalog, and one to execute operations. The full catalog stays internal and is searched at runtime, so your model's context stays clean.

To upgrade, simply remove all service flags from your MCP configuration.

Read the full announcement: appwrite.io/blog/post/announcing-appwrite-mcp-server-2
